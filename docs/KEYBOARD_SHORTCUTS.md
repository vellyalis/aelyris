# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+P | Command Palette |
| Ctrl+P | Quick Open (file search) |
| Ctrl+, | Settings |
| Ctrl+Shift+O | Open Folder |
| Ctrl+N | New File |

## Terminal

| Shortcut | Action |
|----------|--------|
| Ctrl+` | Focus Terminal |
| Ctrl+Shift+T | New Terminal Tab |
| Ctrl+Shift+W | Close Terminal Tab |
| Ctrl+Tab | Next Tab |
| Ctrl+Shift+Tab | Previous Tab |
| Ctrl+F | Search in Terminal |

## Editor

| Shortcut | Action |
|----------|--------|
| Ctrl+S | Save File |
| Ctrl+W | Close File |

## Search

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+F | Toggle Search Panel |
| Ctrl+Shift+E | Close Search Panel |

## Agent

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+A | Start Agent |
| Ctrl+[ | Previous Agent Session |
| Ctrl+] | Next Agent Session |
| Ctrl+0-9 | Switch to Agent Session N |
