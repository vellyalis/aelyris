## 色彩・デザイン詳細

### 背景
- メイン背景: 純黒に近い `#0d0d0d` 〜 `#141414`
- サイドバー: わずかに明るい `#1a1a1a`
- セッションカード背景: `#1e1e1e` 程度
- セッションカードボーダー: 金/琥珀色 `#c8a050` 系（アクティブ時）

### テキスト
- メインテキスト: 白 `rgba(255,255,255,0.88)`
- 二次テキスト: `rgba(255,255,255,0.5)`
- 薄いテキスト: `rgba(255,255,255,0.3)`
- ステータス "Idle": 緑ドット `#4ade80`
- ステータス "Edit": 黄色
- コスト表示: `<$0.01` 薄いグレー

### Project Header Bar
- 高さ: 約 30-35px
- 左: プロジェクトアバター（ピクセルアート、16x16程度）
- プロジェクト名: 太字白文字
- ブランチ: `⚡main` アイコン付き、薄い色
- ステータス: `● Idle` 緑ドット + テキスト
- 右端: モデル情報 `Opus 4.6 (1M context)` + コスト `<$0.01`

### Sessions パネル（右）
- ヘッダー: "Sessions" と "Activity" のタブ切替（太字/通常で区別）
- 右上: 通知ベル + 追加ボタン
- カード: 角丸6-8px、金色ボーダー（1-2px）、暗い背景
- カード内: アバター + 名前 + ブランチ + プログレスバー + パーセント
- カード下: ステータス + 経過時間 + モデル + コスト

### Toolkit パネル（右下）
- 2列グリッドレイアウト
- 各ボタン: テキスト + アイコン + 右端に色付きバッジ
- バッジ色: 紫、緑、赤、青等（機能カテゴリを示す）
- ヘッダーに "Toolkit" + プロジェクト選択

### 左サイドバー
- 上部: プロジェクトディレクトリのファイルツリー
- VS Codeと同じ折りたたみ式（▶/▼）
- ファイルアイコン付き
- 下部: 紫色背景のエリア（Helm = タスク/ロードマップ）

### 下部ステータスバー
- ワークスペースタブ（セッション切替）
- `◀ reply` ボタン
- 各タブにプロジェクトアイコン

## フォント
- UI: IBM Plex Sans
- ターミナル/コード: IBM Plex Mono  
- ノート: Monaspace Neon Var
- タイトル: Archivo Variable

## 重要なインタラクション
- `⌘0-9`: セッションジャンプ
- `⌘[ / ⌘]`: 前/次セッション
- Toolkit: ワンクリックアクション実行
- セッションカードクリック: そのセッションにフォーカス
